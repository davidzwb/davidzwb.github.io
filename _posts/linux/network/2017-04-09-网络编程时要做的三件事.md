# SO_REUSEADDR

使得一个 socket 不用再等待前一个 socket 的 TIME_WAIT 状态，而可以直接 bind。

# SIGPIPE

忽略 SIGPIPE signal，当向一个已关闭的 socket 写入时，会收到 SIGPIPE signal，默认行为是关闭整个进行，所以要将它忽略。

# TCP_NODELAY

开始这个选项会使用 Nagle's Algorithm，造成延迟增加。
